### ✏️Explanation✏️
네이버 스포츠 > 야구 > 일정/결과 > 경기 기록 페이지를 크롤링해 
경기 결과, 선수단 기록 등을 이미지로 저장한 후 자신의 tistory 블로그에 자동으로 포스팅합니다.
